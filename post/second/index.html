Comparing Hello World Programs

By Hariharan Srinath under golang python

Hello World programs are the first set of programs typically introduced in any programming tutorial. Let’s explore Hello World programs across a few languages.

Go Language

The Go Language released by Google is notable for its C Language inspired syntax (indeed Ken Thompson is a founder) that nevertheless simplifies a lot of boilerplate and introduces a number offeatures geared to writing modern and fast web server applications with the standard library itself.

In this Hello World program, we import the fmt standard library which contains most of the formatted output functions and use Println() to print Hello World

import "fmt"

func main(){
    fmt.Println("Hello World")
}

Python 2 Hello World

Python is a very simple to learn and use interpreted language with a focus on simplicity. The guiding philosophy is There should be one and only one way of doing something. The Hello World program is self-explatory

print "Hello World"

Python 3 Hello World

Python 3 is the modern and incompatible evolution of the Python Language. One change you will immediately note is that print is now a function rather than a stagement

print("Hello World")
